# JuxeBot
Juxebot is a discord music bot programmed in Python using the wavelink library.

# Required files

## Lavalink

Install the Lavalink jar file from [freyacodes/Lavalink](https://github.com/freyacodes/Lavalink)

## Py-Cord

Install the pycord library using pip in your terminal

`python3 -m pip install pycord`

## youtube_dl

Install the youtube_dl library using pip in your terminal

`python3 -m pip install youtube_dl`

## wavelink

Install the wavelink library using pip in your terminal

`python3 -m pip install wavelink`

#Running the bot

Run the lavalink file in your terminal using screen or open a second tab or terminal for the python file.

`java -jar Lavalink.jar`

`python3 main.py`
